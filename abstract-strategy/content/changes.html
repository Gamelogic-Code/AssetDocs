Changes

Version 3.1.0

🎉 New Features

  • Added features for advanced graph debugging.

🐞 Bug Fixes

  • Renamed EmptyCellCondition to script to match the name of the class. This prevented this node from being serialized correctly in graphs.

🔧 Engineering

  • Fixed many inspection warnings.
  • Deprecated misnamed events in MoveManager<TPoint, TPieceSettings>:
    • onMoveStart in favor of OnMoveStart
    • onMoveEnd in favor of OnMoveEnd
    • onAllMovesEnd in favor of OnAllMovesEnd
  • Deprecated misnamed events in TurnManager<TPoint, TPieceSettings>:
    • onPlayerChange in favor of OnPlayerChange
    • onTurnStateChange in favor of OnTurnStateChange
    • onPlayerVictoryStateChange in favor of OnPlayerVictoryStateChange
  • Deprecated misnamed events in Player<TPoint, TPieceSettings>:
    • onVictoryStateChanged in favor of OnVictoryStateChanged
  • Deprecated misnamed properties in IPieceProperties:
    • pieceID in favor of PieceID
    • uniqueID in favor of UniqueID
    • playerID in favor of PlayerID
  • Deprecated HumanPlayer<TGameState, TPoint, TPieceSettings> in favor of HumanPlayer<TPoint, TPieceSettings>.

📄 Documentation

  • Improved the topic documentation, in particular updated the documents to reflect changes from Grids 1 to Grids 2.
  • Fixed typos and bugs in the API documentation.

🧪 Samples

  • Added AI for Tic Tac Toe and Reversi that got lost.

⬆️ Upgrade Notes

  • If you use events in MoveManager<TPoint, TPieceSettings>, you need to rename them as described above.
  • If you use events in TurnManager<TPoint, TPieceSettings>, you need to rename them as described above.
  • If you use properties in IPieceProperties, you need to rename them as described above.
  • If you used the EmptyCellCondition node in a graph, you need to re-add it since it would not have saved before.
  • If you use HumanPlayer<TGameState, TPoint, TPieceSettings>, you need to remove the TGameState type parameter.

Version 3.0.0

🔧 Engineering

  • Updated to work with Unity 2019.4.14 LTS
  • Updated to work with Gamelogic Extensions 3.0.0.
  • Restructured as a package.
  • Removed all warnings.
  • General small code improvements (formatting, naming, etc.).
