Advanced Graph Debugging

Additional debugging tools are available for advanced users to help diagnose issues with the BehaviourTree graphs. These features can be accessed by adding the GAMELOGIC_GRAPH_DEBUG symbol to the project's scripting define symbols.

When this symbol is defined, the following features are available:

  • BehaviourTree has additional inspector buttons with which the nodes embedded in the asset can be made visible or invisible. This can be useful for validating that nodes serialize correctly (especially if you defined custom nodes.)
  • Hidden serialized fields of the BehaviourTree will become visible in the inspector.
  • Hidden serialized fields of the BTNode will become visible in the inspector.
