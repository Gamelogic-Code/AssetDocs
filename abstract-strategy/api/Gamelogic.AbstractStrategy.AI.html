Namespace Gamelogic.AbstractStrategy.AI

Classes

BTNode

The base class for all nodes in a BehaviourTree.

BehaviourTree

A class the represents a computational graph. Each node in this graph takes some inputs, and calculates outputs, that can in turn be fed as inputs into other nodes.

All nodes produce outputs as lists. When a node has multiple inputs, these are all combined into a single list for the node to operate on.

Blackboard

Blackboard object, unique to each agent using a behaviour tree, that contains information for the tree's operation.

CanMakeLine

A conditional node to test whether a given player can complete a line with only one placement.

CompositeNode

A composite node in the behaviour tree has multiple children and ticks them in some order.

ConditionalNode

A conditional node is a leaf node that checks for some gameManager state condition and returns Success if it is met, or Failure if not

DecoratorNode

A decorator node has one child and modifies its return result in some fashion.

EmptyCellCondition

A conditional node to test the contents of a cell or a collection of cells on the grid. The last tested node optionally saves its position into pointVariable in the blackboard for other nodes to operate on

GraphEditorAttribute

An attribute used to decorate behaviour tree nodes. Apply this to custom behaviours to have them appear in the behaviour tree editor list.

GridAIPlayer

AI player for grid type gameManager states, that uses behaviour trees

Interval

A class for representing an interval using a start point, and a width. It is essentially a 1D rectangle.

Inverter

The inverter node ticks its child and negates the child's return result.

MemorySelector

A selector node ticks all of its children sequentially, until one of them returns Success, Running or Error. If all children return Failure, then it will return that too. Additionally, as long as this node remains open, whenever a child returns Running, on subsequent ticks execution passes directly to that child instead of ticking previous children again.

MemorySequence

A sequence node ticks all of its children sequentially, until one of them returns Failure, Running or Error. If all children return Success, then it will return that too. Additionally, as long as this node remains open, whenever a child returns Running, on subsequent ticks execution passes directly to that child instead of ticking previous children again.

NodeBlackboard

Node specific blackboard, for node info.

PlacePiece

Place a piece at a position. Will return Success if the move is valid or Failure if it is not.

PlacePieceRandomly

Place a piece at a position.

RootNode

Class for the root node of the behaviour tree. Has exactly one child and never has a parent. Cannot be removed from the tree, and doesn't perform any action except simply ticking its child repeatedly.

Selector

A selector node ticks all of its children sequentially, until one of them returns Success, Running or Error. If all children return Failure, then it will return that too.

Sequence

A sequence node ticks all of its children sequentially, until one of them returns Failure, Running or Error. If all children return Success, then it will return that too.

TestCellCondition

A conditional node to test the contents of a cell or a collection of cells on the grid. The last passed node optionally saves its position into pointVariable in the blackboard for other nodes to operate on
