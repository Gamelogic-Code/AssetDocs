TileCell Methods

The TileCell type exposes the following members.

Methods

  Name Description
__UpdatePresentation
This methods is called by the editor, and is to update the cell representation to it's current state. (Typically, this is necessary for serialized fields that affect the presentation.For example, the color may be saved, but the actual object may reset it's color).
AddAngle
Adds the given angle to the current angle of the tile. If the tile is in the XY plane, it will be added to the Z angle.
OnDrawGizmos
Draws the Gizmos for this tile: a label with the tile name. It is often convenient, for this reason, to make the tile name the coordinate of the tile.
SetAngle
Sets the angle of the tile. If the tile is in the XY plane, the angle will be the Z angle.
