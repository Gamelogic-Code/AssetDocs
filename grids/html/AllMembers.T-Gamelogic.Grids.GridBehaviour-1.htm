GridBehaviour<(Of <(<'TPoint>)>)> Members

The GridBehaviour<(Of <(<'TPoint>)>)> type exposes the following members.

Methods

  Name Description
InitGrid
When this behaviour is attached to a grid builder, this method is called once the grid is created, and all cells(tiles) have been instantiated. Override this to implement custom initialisation code. (You can access the grid through the Grid property).

Properties

  Name Description
Grid
The grid data structure, containing cells as elements. (It's likely that this will return a grid of a different (more general) cell type in the future).
GridBuilder
Returns the grid builder attached to the same game object as this grid behaviour. (It's provided, but you will mostly need only the Grid and Map.)
Map
The map used to build the grid.
MousePosition
Get's the mouse position in Grid coordinates. Currently it is only useful for 2D grids, rendered with orthographic cameras.
