Dither Texture Bias

Built-in Only

Turns the image into a dithered version using a user-provided texture pattern. The textureâ€™s values act as a bias added to the image before quantization, producing a stylized dither effect. Use this effect when you want full artistic control over the pattern (Bayer, blue noise, etc.).

The alpha channel is not changed by this effect.

Tip
  • A neutral pattern (flat gray) produces no dithering.
  • Bayer, blue noise, and photographic film-style patterns give very different looks.
  • When using high smoothness values, subtle dithering patterns can act as a film-grain-like texture.
