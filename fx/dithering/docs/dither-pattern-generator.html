Dither Pattern Generator

The Dither Pattern Generator is one of the texture generation editor tools that allows you to create custom dither patterns directly inside Unity, suitable for use in dithering shaders such as Dither Texture Bias.

Open it via:

Gamelogic → Tools → Texture Generation → Dither Pattern Generator

Overview

The tool supports two output modes:

  • Greyscale — produces a single-channel dither matrix (monochrome dithering).
  • RGB — produces independent matrices for Red, Green, and Blue (color dithering).

These textures are suitable to use with texture bias dithering shaders such as these:

  • Post Effect Dither Texture Bias
  • Object Shader Dither Texture Bias

Each mode offers its own configurable matrix and controls.

How It Works

  1. Select the Pattern Type (Greyscale or RGB).
  2. Edit the Matrix Values.
  3. Review the Preview (updates automatically).
  4. Click Save Texture to export the dither pattern as a PNG.

Pattern Types

Below are the supported pattern types and their editable properties.

Greyscale

Generates a single-channel dither pattern from one matrix. This is ideal for classical monochrome Bayer matrices, blue-noise patterns, or hand-crafted greyscale masks.

Control Range Description
Matrix floats Editable grid of floating-point values used as the pattern.
Normalize On/Off Normalizes matrix values to [0, 1] before preview/export.
Resolution Integers ≥ 1 Pixel size of each matrix cell in the preview and output.

RGB

Generates a full-color dither pattern using three independent matrices—one for each color channel. This allows patterns with channel-separated structure, useful for stylized color dithering, retro console effects, or print-style halftone patterns.

Control Range Description
Matrix R floats Matrix defining the dither pattern for the Red channel.
Matrix G floats Matrix defining the dither pattern for the Green channel.
Matrix B floats Matrix defining the dither pattern for the Blue channel.
Normalize On/Off Normalizes each channel matrix independently to [0, 1].
Resolution Integers ≥ 1 Pixel size of each matrix cell in the preview and final PNG.
