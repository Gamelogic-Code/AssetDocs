Schemes for orienting hatch layers.

public enum OrientationScheme

Fields

Fixed = 0

Relative to the first layer, layers are rotated by a fixed angle.

That is:

  • Layer 0: 0 degrees
  • Layer 1: 90 degrees
  • Layer 2: 45 degrees
  • Layer 3: 135 degrees
  • Layer 4: 22.5 degrees
  • Layer 5: 112.5 degrees
  • Layer 6: 67.5 degrees
  • Layer 7: 157.5 degrees

After this the pattern repeats.

Golden = 2

The angles are multiples of 180 times the golden ratio (approximately 1.61803398875).

This is the best compromise between maximizing the angle difference between any present layers, and being the same regardless of the number of layers. No two layers will ever overlap exactly.

Parallel = 3

The layers are all aligned.

Uniform = 1

The angles are uniformly distributed between 0 and 180 degrees according to the number of layers.

For example for 3 layers, the angles would be:

  • Layer 0: 0 degrees
  • Layer 1: 60 degrees
  • Layer 2: 120 degrees
