Mixbox

Some effects require the Mixbox plugin. Mixbox provides a color-mixing model that more closely mimics the way pigments mix, and is therefore useful for implementing effects meant to look like drawing or painting.

  • The effects that require the plugin will only be available if the plugin is installed.
  • To install the plugin:

Open Window | Package Manager and choose + | Add package from git URL...: https://github.com/scrtwpns/mixbox.git#upm

Effects that use Mixbox require that the MixboxLUT texture is set on the effect properties. (This encodes the nonlinear information used to mix colors.)

Note that Mixbox has its own licensing terms, so check that out before you use it.
