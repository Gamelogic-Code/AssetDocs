Convex Hull Hatch

Maps each pixelâ€™s color toward a set of primary colors using iterative convex-hull projection, while applying layered cross-hatching to modulate how strongly each primary influences the result. This produces stylized, textured palette mapping that blends palette reduction with drawing-like shading.

Hatching is applied across multiple layers, each rotated according to the selected orientation scheme, allowing effects ranging from classic engraving to organic irregular cross-hatching.
