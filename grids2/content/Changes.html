Changes

Version 4.1.0

🎉 New Features

  • Added a new flag on grid builders to force recreation of cells at runtime. This is to allow for grids that may change shape each time they are computed.
  • Added implicit conversions between GridPoint2 and Vector2Int, and between GridPoint3 and Vector3Int.
  • Added explicit conversion from Vector2 to GridPoint2, and from Vector3 to GridPoint3.
  • Made it possible for nodes to specify their width (see Width ), and made nodes with Gamelogic.Grids2.Graph fields wider. Nodes also change their width if they don't match the Width property.
  • Made it possible for GraphWindows to specify the order of groups by implementing the GetOrder method.

🐞 Bug Fixes

  • Improved GraphPropertyDrawer so that it does not throw Stack Empty errors in Unity 6000.
  • Fixed a NullReferenceException in GridBuilder<TCell> when the shape graph has not been assigned.
  • Made MultiLayerShapeNode, OrthographicProjectionShape3Node and SimpleLayerShapeNode recompute their layers when they are recomputed. (This would only affect the results if any of the layers were dynamic),
  • Shape1Graph, Shape2Graph, Shape3Graph, and SpaceMapGraph do not automatically recompute their shapes each time GetShape is being called. Before, dynamic grids could not be properly be aligned because the shape used to create the grid and the one used for alignment were different.

Version 4.0.0

🎉 New Features

  • It is now possible to change the name of all nodes in graphs.

🎨 UI

  • Improve the look and feel of the graph editor windows.
  • Shape and space map graphs in the inspector now have Create / Edit buttons to open the graph editor.

🧪 Samples

  • Added a new sample: 3D Tiled Grid that shows how to use custom hex cells with grid builders, and how to use ray casting to support grid mouse input.

🔧 Engineering

  • MakeNewGraph returns null if the graph was not saved.
  • Added type parameters to GraphPropertyDrawer.

⬆️ Upgrade Notes

  • If you use GraphPropertyDrawer, specify the new additional type arguments.
  • If you use MakeNewGraph, check whether the result is null and handle the case appropriately.
