Remarks

Grid points work like vectors, except their components are integers. Grid points can be used on their own, but they are frequently used with grids and shapes.

 <xref href="Gamelogic.Grids2.GridPoint2" data-throw-if-not-resolved="false"></xref> was introduced before Unity's <xref href="UnityEngine.Vector2Int" data-throw-if-not-resolved="false"></xref>, and serve the same purpose. The types are
 implicitly convertible.
